大会入口

このページから各大会に進むことで、すべての大会の管理者の権限を行使できます。システム管理者のみ利用できます。

このシステムでできること

このシステムは大会の管理システムです。

ユーザが大会に参加するとき、新規登録から次のことができます。

  1. 聴講者として大会に参加【自動承認・非自動承認】
  2. 講演者として大会に参加し、発表する論文の投稿・管理を行う【非自動承認】

システム管理者から承認された各大会の管理者は次のことができます。

  1. 各大会の管理

管理者メニューについて

管理者トップページ（大会入り口ページ）：

このページです。このページから各大会に進むことで、すべての大会の管理者の権限を行使できます。システム管理者のみ利用できます。

大会追加ページ：

大会の追加ができます

大会一覧ページ

大会の基本情報の編集と大会の削除、メール送信機能などができます。システム管理者のみ利用できます。

  1. 大会ごとのメール送信ページ：大会ごとに登録者全員に対してメールを送信することができます。
  2. 大会確認ページ：大会の基本情報の編集、大会の削除ができます。

一般ユーザ一覧ページ

一般ユーザの一覧から、ユーザの編集と削除ができます。システム管理者のみ利用できます。
